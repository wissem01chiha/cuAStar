Detailed Description

template<typename T>
class Node2d< T >

Base class for 2D nodes representation.

Template Parameters
TNumeric type for the sum_cost (e.g., T, float)
Parameters
x: x-coordinate in the grid
y: y-coordinate in the grid
z= 0 : this coordinate is used only to ensure generic function call that do not throw an error in using 3d nodes specific operations to 2d nodes based template methods.
sum_cost: cumulative cost from the start node to the current node
p_node: pointer to the parent node
